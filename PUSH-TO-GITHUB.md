# 🚀 Push to GitHub Instructions

## Quick Steps:

### 1. Create GitHub Repository

1. Go to: https://github.com/new
2. Repository name: `escape-the-nightmare` (or any name you want)
3. Description: `Horror-themed escape room puzzle game with mobile PWA support`
4. Keep it **Public** (so Vercel can access it)
5. **DO NOT** check "Add README" or "Add .gitignore" (we already have them)
6. Click **"Create repository"**

### 2. Push Your Code

After creating the repo, GitHub will show you commands. Use these:

```bash
git remote add origin https://github.com/Aweskouki/escape-the-nightmare.git
git branch -M main
git push -u origin main
```

Or run this single command (replace YOUR_REPO_NAME):

```bash
git remote add origin https://github.com/Aweskouki/YOUR_REPO_NAME.git && git branch -M main && git push -u origin main
```

### 3. Done!

Your code is now on GitHub! ✅
