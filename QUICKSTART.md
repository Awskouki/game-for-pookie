# Quick Start Guide

## 🚀 Get Started in 3 Steps

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Game
```bash
npm run dev
```

### 3. Open Your Browser
Navigate to: `http://localhost:3000`

## 🎮 First Time Playing?

### Controls
- **WASD** or **Arrow Keys** = Move & Jump
- **Space** = Jump (alternative)
- **ESC** = Pause

### Your First Level
1. **Move right** to start exploring
2. **Collect coins** (yellow circles) for points
3. **Avoid enemies** (orange/red squares)
4. **Grab power-ups** (floating colored orbs)
5. **Solve puzzles** at gates to unlock them
6. **Reach the end** to complete the level!

### Tips for Success
- 🎯 **Chain collectibles** quickly for combo bonuses
- 💪 **Use power-ups** strategically
- 🛡️ **Shield protects** you from one hit
- 💡 **Hints available** after 2 wrong answers
- ⚡ **Double-jump** helps reach high places

## 🐛 Troubleshooting

### Game won't start?
- Make sure Node.js v20+ is installed
- Run `npm install` first
- Check console for errors

### No sound?
- Click the speaker icon to unmute
- Check browser audio permissions

### Progress not saving?
- Ensure localStorage is enabled in your browser
- Check browser privacy settings
